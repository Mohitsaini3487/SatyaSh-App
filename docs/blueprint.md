# **App Name**: SatyaSh शोधak

## Core Features:

- Multi-Method Authentication: Secure user authentication supporting email, username, and mobile number logins.
- Comprehensive User Profiles: Detailed profile section allowing users to manage personal information, social links, preferences, and security settings.
- Real-Time Metrics Dashboard: Dynamic dashboard displaying news articles analyzed, fake news detection count, active users, and supported languages.
- AI Chatbot: 24/7 multilingual AI chatbot for user support and guidance, localized for India's top 10 languages.
- Fraud Detection: Analysis tool leveraging machine learning and the Gemini 2.5 Flash API to detect fake news in text, files, links, and videos, providing a fraud percentage and detailed explanation.
- Real-Time News Feed: Curated news feed delivering educational content across various categories, updated every 5 minutes.
- AI Summarization Tool: A summarizer tool that leverages AI to shorten text, file, link and URL inputs, that offers output in the user's selected language. The LLM tool determines if/when it is appropriate to summarize each document based on document type and other quality considerations.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a trustworthy and intelligent feel.
- Background color: Light grayish-blue (#E8EAF6) to maintain a professional and calm atmosphere.
- Accent color: Vibrant magenta (#E91E63) to highlight key interactions and calls to action.
- Body font: 'PT Sans' for readability and a modern feel; Headline font: 'Space Grotesk' for emphasis.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use detailed and informative icons, focusing on clarity and multilingual accessibility.
- Implement smooth transitions and subtle animations to enhance user engagement.
- Full-screen animations on the login page with a detective aesthetic.
